# 📝 Comandos Básicos do Git

## Configuração inicial
- `git config --global user.name "Seu Nome"` — define o nome do autor
- `git config --global user.email "seu@email.com"` — define o e-mail

## Criando e versionando
- `git init` — inicia um repositório na pasta atual
- `git status` — mostra o estado dos arquivos
- `git add <arquivo>` — envia o arquivo para a staging area
- `git add .` — adiciona todos os arquivos modificados
- `git commit -m "mensagem"` — registra as mudanças no histórico

## Consultando
- `git log` — mostra o histórico de commits
- `git log --oneline` — histórico resumido (uma linha por commit)

## 💡 Aprendizado prático
No terminal, silêncio geralmente é sucesso: se o comando não retorna
mensagem, ele funcionou. E cuidado ao colar comandos no PowerShell —
sempre um por vez!
