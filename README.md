# 🚀 Desafio Git e GitHub — DIO

Repositório criado como parte do desafio prático de **Git e GitHub** da [DIO](https://www.dio.me/), com o objetivo de praticar a criação, atualização e sincronização de repositórios.

## 📋 O que foi praticado

- Criação de repositório local com `git init`
- Versionamento com `git add` e `git commit`
- Conexão com repositório remoto no GitHub
- Sincronização com `git push` e `git pull`
- Trabalho com branches e merge

## 📁 Estrutura do repositório

- `anotacoes/` — anotações das aulas
- `exercicios/` — registro dos comandos praticados
